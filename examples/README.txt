File EEG di esempio da provare con "LOAD FILE" in SOGNO_CANE.

  example_human_8s.csv   8 s di EEG umano simulato (8 canali, 250 Hz)
  example_dog_8s.csv     8 s di EEG di cane simulato (8 canali, 250 Hz)

Come usarli:
  1. Avvia SOGNO_CANE.
  2. In un pannello (HUMAN o DOG) premi LOAD FILE e scegli uno di questi file.
  3. Premi START: il file viene riprodotto e tradotto in MIDI.

Formati supportati da LOAD FILE: .csv .tsv .txt .edf .bdf .rec .npz .npy

Puoi anche scaricare EEG reali in formato EDF/BDF, ad esempio dai file di test
pubblici di PhysioNet (https://physionet.org) o BioSemi
(https://www.biosemi.com/download/BDFtestfiles.zip).
